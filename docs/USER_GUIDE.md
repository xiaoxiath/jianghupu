# 欢迎来到《江湖残卷》 - 玩家指南

## 1. 游戏简介

欢迎来到《江湖残卷》的世界！

这是一款与众不同的武侠游戏。这里没有固定的剧本，没有千篇一律的任务线。你的故事，由一个强大的 **AI “说书人”** 实时为你谱写。你的每一个选择，都可能改变你自己的命运，甚至整个江湖的格局。

准备好进入一个真正“活”着的、充满变数的武侠世界，开启一段完全属于你的传奇了吗？

## 2. 如何开始游戏

*（此部分将在游戏发布 V1.0 版本时填充）*

1. 从 [发布页面](https://github.com/xiaoxiath/jianghupu/releases) 下载最新的发行版 (`.zip` 或 `.tar.gz`)。
2. 将压缩包解压到你喜欢的任意位置。
3. **Windows**: 双击运行 `jianghupu.exe`。
4. **macOS / Linux**: 打开终端，进入解压后的目录，然后运行 `./jianghupu`。

## 3. 核心概念

理解以下几个概念，将帮助你更好地闯荡江湖：

### AI 说书人

他是你的专属故事讲述者。你说书人会根据你的行为、你角色的状态、以及世界上正在发生的大事（比如某个门派被灭了），来为你描述周围的环境，并提供你接下来可以做的选择。

游戏内置了多种不同性格的说书人（如“宿命”、“诙谐”等），未来你将有机会切换他们，体验不同风格的江湖故事。

### 轮回与传承 (Roguelike)

在这个江湖中，死亡是常态，但并非终点。

当你的一生结束后，你的事迹（无论是名震天下还是默默无闻）都会化为世界的历史。你可以创建一名新角色，开启一次**新的轮回**。

在新的轮回中，你可能会发现前代角色留下的“遗产”——或许是一本失传的秘籍，或许是一笔神秘的宝藏，又或许，只是一个在江湖中流传的、关于你的传说。

### 活态世界

你不是这个世界的中心。

在你修炼、探索、行侠仗义的同时，这个世界也在自行演化：

- **NPC 们**在各自修炼，他们的实力会随着时间增长。
- **各大宗派**之间的关系在动态变化，昨天还亲如兄弟的两个门派，今天可能就反目成仇。
- **重大事件**会随机发生，比如魔教入侵、武林大会等，这些都可能波及到你。

请时刻留意世界的变化，因为它随时可能给你带来机遇，或是危机。

## 4. 基本操作

- **做出选择**: 大多数时候，游戏会以选项的形式呈现在你面前。你只需要输入选项前的数字或字母，然后按回车即可。
- **输入指令**: 当你看到 `你的选择是？` 提示中包含 `输入指令 (save/load/exit)` 这样的选项时，选中它，你就可以输入更复杂的游戏指令。目前支持的指令有：
  - `save [槽位号]`: 保存当前游戏进度到一个指定的槽位（例如: `save 1`）。
  - `load [槽位号]`: 从一个指定的槽位读取游戏进度（例如: `load 1`）。
  - `exit`: 退出游戏。

## 5. Mod 系统：打造你的专属江湖

### 什么是 Mod?

Mod (Modification) 是由社区玩家制作的内容扩展包。它们可以为游戏增加新的事件、物品、武学、宗派、NPC，甚至是全新的玩法和故事线。

### 如何安装 Mod?

1. 在你的游戏根目录下，找到名为 `mods` 的文件夹。如果它不存在，请手动创建一个。
2. 将你下载的 Mod 文件夹（例如，一个名为 `cool-sect-mod` 的文件夹）完整地放入 `mods` 目录内。
3. 重新启动游戏。游戏会自动加载所有 `mods` 文件夹中的 Mod。

**目录结构示例:**

```
jianghupu/
├── mods/
│   ├── cool-sect-mod/
│   │   ├── manifest.json
│   │   └── ...
│   └── another-event-pack/
│       ├── manifest.json
│       └── ...
└── jianghupu.exe
```
