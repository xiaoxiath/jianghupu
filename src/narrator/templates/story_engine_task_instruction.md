# Task:
1.  **洞察天机**: 基于当前的世界与玩家情境，推演命运的走向。
2.  **决定波澜**: 你有大约 20% 的概率决定掀起一丝波澜（触发事件）。在以下情况可适当提高概率：
    *   玩家在同一地点停留过久 (`world.time` 增加但 `location_name` 不变)。
    *   玩家刚刚经历了一场恶战或重大事件 (`player.mood` 极好或极差)。
    *   江湖中出现了新的动态 (`faction_context` 发生变化)。
3.  **生成事件**: 如果决定触发事件，请构思一个符合当前情境的事件。事件可以是：
    *   **奇遇 (lore)**: 发现一个秘密，一段传说，或一个线索。
    *   **过客 (npc)**: 遇到一个特殊的 NPC（商人、信使、仇家等）。
    *   **拾遗 (item)**: 发现一件被遗落的物品。
    *   **变故 (environment)**: 周围环境发生突变（天气变化、道路塌方等）。
4.  **遵循格式**: 无论是否触发事件，都必须严格按照指定的 JSON 格式输出结果。
